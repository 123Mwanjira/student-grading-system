# Student Grading System (JavaScript)

##  Overview
The Student Grading System is an interactive application that evaluates student marks and assigns grades based on predefined criteria. It demonstrates core JavaScript concepts such as input handling, validation, conditional logic, and continuous execution in a command-line environment.

---

## ✨ Features
- Accepts user input for student marks  
- Validates input to prevent invalid entries  
- Automatically assigns grades based on marks  
- Runs continuously until the user chooses to exit  
- Simple and user-friendly command-line interface  

---

##  Technologies Used
- JavaScript (ES6)  
- Node.js  
- prompt-sync
